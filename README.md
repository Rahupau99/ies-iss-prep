# ies-iss-prep.github.io
How to use this library (Offline-Use) 
1 Keep all HTML files in the same folder on your device — this index links to them by filename.
2 Open index.html in Safari or Chrome  and add it to your home screen for instant access.
3 Progress for each paper is saved separately in your browser — it won't mix across years or subjects.

How to Use
1 Open any paper and go to the Practice tab.
2 Write your answer in the text area — match the word limit.
3 Tap Show Model Answer to see key concepts, full answer & examiner tips.
4 Rate yourself — Good / Partial / Needs Work. Progress saves automatically.
5 Use the Mock tab for a timed 3-hour full-paper simulation.
